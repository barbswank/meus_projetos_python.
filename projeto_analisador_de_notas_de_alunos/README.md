# Analisador de Notas de Alunos

Projeto simples em Python para analisar notas de alunos.

O programa calcula:

- Média da turma
- Alunos acima da média
- Alunos reprovados
- Dobro das notas usando Lambda
- Ordenação dos alunos por nota

## Conceitos utilizados

- Listas
- Dicionários
- List Comprehension
- Lambda
- Map
- Estruturas de controle

## Objetivo

Praticar lógica de programação e manipulação de dados em Python.
