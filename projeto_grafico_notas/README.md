# Projeto Gráfico de Notas

Este projeto cria um gráfico de barras com as notas de alunos usando Matplotlib.

## Como rodar
1. Instale o matplotlib: `pip install matplotlib`
2. Execute o arquivo: `python grafico_notas.py`

## Conceitos usados
- Listas
- Bibliotecas externas (Matplotlib)
- Loops implícitos no Matplotlib
