# Projeto: Manipulação de Strings

Este projeto contém exercícios práticos de manipulação de strings em Python:

## Conceitos abordados

1. **REPLACE**
   - Substituir partes de uma string por outras ou remover caracteres
2. **JOIN**
   - Transformar listas em strings usando separadores
3. **SPLIT**
   - Transformar strings em listas usando separadores
4. **SLICE**
   - Pegar partes (fatias) de strings usando índices

## Como usar

- Execute cada arquivo separadamente no terminal:
```bash
python replace.py
python join_split.py
python slice.py
